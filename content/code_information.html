Code information¶

This module needs Python 2.6 or later.

class pyrotrfid.Bacteria(annotation_string)[source]¶

A bacteria is described by its taxon. This can be determined by parsing an entry in the Green Genes database. A bacteria object can be used as a key to a dictionary. The annotation_string typically looks something like:

239015_EU071527.1_and_resistance_microorganisms_European_clean_room_ESTEC_HYDRA_facility_clone_EHFS1_S15a_k__Bacteria;_p__Proteobacteria;_c__Gammaproteobacteria;_o__Xanthomonadales;_f__Xanthomonadaceae;_g__Thermomonas;_Unclassified;_otu_4045

The abbreviation OTU stands for “Operational Taxonomic Unit”.

name[source]¶

Return a short and human readable name.

class pyrotrfid.Fragment(length, bacteria, ref, read_length, sw_score)[source]¶

Once a short read is digested, a restriction fragment of a certain size is created. It has a length, is associated to a specific bacteria, contains the reference of the short read from which it came and the mapping quality of that short read.

Parameters:
  • length (int) – Length of the restriction fragment.
  • bacteria (Bacteria) – Taxonomic information.
  • ref (str) – The reference name of the read.
  • read_length (int) – The length of the read.
  • sw_score (int) – The Smith-Watermann score of the alignement of the original read.

class pyrotrfid.Peak(length)[source]¶

All restriction fragments of the same size are regrouped in one peak object. Hence a peak object has a unique peak length and contains many fragments. Furthermore, inside a peak, fragments coming from the same bacteria are grouped.

class pyrotrfid.Sample(path, min_frag_size, max_frag_size, hard_lag)[source]¶

Every raw file containing short reads is considered as a separate sample. A sample will be converted to FASTAQ format, mapped, sorted and indexed; after which it will be virtually digested, the resulting fragments counted and grouped by bacteria. Plots will be made comparing these results with the results from the wet lab TRFLP.

Parameters:
  • path (str) – Path to the file containing the short reads. Can be SFF or FASTA.
  • min_frag_size (int) – Restriction fragments shorter than this are ignored.
  • max_frag_size (int) – Restriction fragments larger than this are ignored.
  • hard_lag (int) – If not False, the lag will be fixed to this value.

digest(enzyme, primer_length, sw_threshold)[source]¶

Virtually digest all the reads in the bam file. The documentation for the AlignedRead object is here: http://www.cgat.org/~andreas/documentation/pysam/api.html

Parameters:

  • primer_length (int) – The amount of base pairs to cut off from every read.
  • sw_threshold (int) – Ignore reads that mapped with a Smith-Waterman score below this.
